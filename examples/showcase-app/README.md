# ClawGuard Shield — Interactive Demo

A visual demo app that lets you type text and see prompt injection detection in real time.

![Screenshot](https://prompttools.co/screenshot-placeholder.png)

## Quick Start

```bash
cd examples/showcase-app
npm install
CLAWGUARD_API_KEY=cgs_your_key npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Features

- Type or paste any text to scan for prompt injection attacks
- See risk scores, severity levels, and matched patterns in real time
- Built-in example prompts (safe + malicious) to try
- Dark theme, responsive design
- Keyboard shortcut: Ctrl+Enter to scan
